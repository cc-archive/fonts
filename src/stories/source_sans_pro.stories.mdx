# Source Sans Pro

Source Sans Pro is a sans-serif typeface created by Paul D. Hunt for Adobe Systems. It is the first open-source font
family from Adobe, distributed under the SIL Open Font License.

It is used in Vocabulary in 3 font weights: regular (400), semibold (600) and bold (700).
